hexo命令

发表于 2021-10-12 阅读次数：

hexo new "article title"

新建一篇文章

hexo g

生成静态网页

hexo d

上传到github

hexo s

本地服务查看页面

hexo clean

网站重新部署后需要执行此命令清理public文件夹内容

hexo new page newPage

创建一个页面，比如hexo new page tags创建tags(标签)页面
